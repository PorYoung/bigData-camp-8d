At the end, we will share our prospects on the future technology challenges and advances of neuromorphic computing.

报告题目：环境中全氟化合物的来源与人体暴露
报  告  人：祝凌燕 教授
报告时间：2016年4月15日上午9:30点到12:00点

报告地点：29教学楼109会议室

主办单位：化学化工学院、校科协

 

报告内容简介：
        由于长期生产使用，全氟烷基酸（perfluoroalkyl acids，PFAAs）在环境和人体中广泛存在，它们具有持久性、生物累积性和生物毒性。全氟辛烷磺酸（perfluorooctanesulfonate，PFOS）和全氟辛酸（perfluorooctanoate，PFOA）是环境中检出频率最高的两种PFAAs。目前欧美等发达国家限制了PFOS和PFOA的生产和使用，但环境中大量的前驱物可通过多种途径降解为PFOS和PFOA。通过对江苏某氟工业园区周边地区、太湖流域水体、华北地区降雪等的野外调查，建立了异构体来源示踪的新方法，揭示了环境中PFAAs的主要来源。定量计算了PFAAs及其典型异构体在人体中的肾清除速率，阐明了PFASs结构特征对其影响规律。利用模型计算，探明人体主要暴露来源其异构体的影响。

 

报告人简介：
        祝凌燕，南开大学环境科学与工程学教授，博士生导师，国家杰出青年科学基金获得者，长江学者特聘教授，曾获中国青年科技奖，中国青年女科学家提名奖，Nature 子刊《Scientific Report》执行编委。本科及硕士毕业于南开大学，博士毕业于新加坡国立大学化学系，曾赴美国印第安那大学公共环境与事物学院从事博士后研究。现任南开大学环境科学与工程学院副院长，天津市城市生态环境修复与污染防治重点实验室副主任。国家自然科学基金委化学部专家评审组成员。主要从事污染生态化学、环境分析化学及污染控制与生态修复等工作。迄今在Environ. Abstract:

摘        要：公开宣告逻辑是研究社会网络信息流动的一种新型逻辑工具，但它存在两点不足：一是不能刻画出宣告者，二是仅适用于全通关系的社会网络。为了克服这两点不足，近5年来，一些国内外学者开始关注带有社会网络的知识流动。目前主要的研究方法有两种：一是阮吉等人在公开宣告逻辑的基础上加入社会网络结构和动态算子；二是谢利民（Jeremy Seligman）、刘奋荣等人借助于混合逻辑的方法考虑社会网络结构。然而，从形式语言上看，上述两种方法都不能解决公开宣告逻辑的两点不足。特别是摩尔公式（Moore Sentence）的存在，使得即使考虑社会网络仍然不能确定一个宣告之后究竟会成立什么样的公式。针对摩尔公式，报告拟从时态认知逻辑的角度来化解这一问题，从而为构造关于社会网络中知识流动的逻辑系统扫清障碍。

Abstract:
        The output regulation problem, also called the servomechanism problem, is to design a feedback control law for a system such that asymptotic tracking for a class of reference inputs and disturbance rejection for a class of external disturbances can be achieved while the stability of the closed-loop system can be maintained.
        In this talk, we will first introduce the output regulation problem, and present some result on the global robust output regulation problem for a class of single nonlinear systems in output feedback form. The concerned systems contain unknown control direction, uncertain exosystem and arbitrarily large parameter uncertainties. Then we will introduce some of our recent result on the cooperative global robust output regulation problem of a class of multi-agent nonlinear systems with unknown control direction via adaptive and robust control approach.

报告题目: Event-Triggered Control for Output Consensus of Heterogeneous Linear Multi-Agent Systems
报  告  人：冯刚教授（香港城市大学）

Abstract:
        In this talk distributed even-triggered control algorithms will be presented for output consensus of heterogeneous multi-agent systems with linear dynamics, with the objective to reduce the number of controller updates and communication exchanges. It is shown that the output consensus problem can be solved by the proposed event-triggered control algorithms if a necessary and sufficient condition is satisfied. Then a self-triggered control scheme is also developed, where continuous monitoring of measurement errors can be avoided. The feasibility of both proposed control schemes is discussed by excluding Zeno behavior. It is also shown that agents are able to achieve output consensus with significant reduction of the number of triggering events, controller updates and communication transmission. As a result, energy can be saved and the lifespan of the agents can be prolonged. A numerical example is given to illustrate the effectiveness of the proposed control schemes.

发言题目：人工智能：非单调推理与信念修正
发言内容：人工智能(AI)是一门综合了计算机科学，生理学和哲学等多种学科和技术的产物。宗旨是设计智能的计算机系统，要求计算机从能够“数据处理”扩展到能够“知识处理”的范畴，特点是让机器会“思考”。信念修正是考虑到新的信息而改变信念的过程，在哲学，数据库和人工智能的领域中为设计理性的Agent导致信念修改的逻辑形式化导致信念修改的研究。知识的不完全性决定了非单调推理的“结论”(通常称为信念)在获得新的知识时，可能被放弃和修正。遵循什么样的公设和如何可行地计算信念修正算子?这导致将描述语言降低到Horn逻辑和采用语言分离技术形成局部信念修正的研究。

发言人简介：张明义，贵州科学院研究员（已退休），1961-1965，贵州大学数学系；1965-1966,贵州省黔东南自治州四清工作队员；1965-1978，贵州省榕江县师范学校教员；1978-1980，贵州大学应用数学研究生；1980-贵州科学院研究人员。其间，1995-2014，西南大学计算机学院兼职教授，1998-现在，贵州大学兼职教授。

发言人简介：詹乃军，博士，中科院软件所研究员，中国科学院大学教授，中国科学院特聘研究员，博士生导师。现任中国科学院软件研究所计算机科学国家重点实验室副主任，中国计算机学会形式化方法专业组秘书长，《Formal Aspects of Computing》、《Journal of Logical and Algebraic Methods in Programming》、《软件学报》、《计算机研究与发展》等编委，及多个国际学术组织任职。主要研究兴趣包括：实时、混成和嵌入式系统，程序验证，模态和时序逻辑，并发计算模型，程序形式语义等。

发言题目：代码验证的理论和工具研究
发言内容：在本报告中，我们将讨论研发可广泛使用的代码验证技术的必要性、可行性和我们的思路。代码验证技术的必要性在于很多形式化方法都是针对抽象模型的，但是抽象模型和代码之间的一致性很难保证，导致这些形式化方法/技术难以应用。代码验证技术的广泛应用的必要性在于形式化方法的社区规模问题。不能广泛使用导致社区越来越小，推广也越来越困难。形式化方法的可行性在于绝大部分程序的逻辑相对简单，程序的Bug通常是因为程序员的疏忽或者考虑不周。我们的思路是充分利用和集成现有的各类程序分析技术，以Scope Logic为基础，以逻辑公式为桥梁，使这些技术之间可以相互辅助，分析得到更多的性质，增强自动化。同时，我们也通过一些自动生成规约的技术来降低代码验证的工作量。
发言人简介：赵建华，1971年生，1998年于南京大学计算机科学与技术系获博士学位，现为南京大学计算机科学与技术系教授,博士生导师。主要研究方向为形式化方法、模型驱动的体系结构、模型检验、程序分析。曾在联合国大学澳门国际软件技术研究所、加拿大Calgary大学访问和合作研究。主持过包括863计划和国家自然科学基金在内的多项国家和省级科研项目。在Formal Aspects of Computing, Journal of Logic and Algebraic Programming, The Computer Journal, Information Processing Letters, Lecture Notes in Computer Science等国际期刊、国际会议和国内一级学报上发表学术论文30余篇。1998年获得教育部科技进步二等奖，2003年获得教育部科技进步二等奖。2005年入选教育部新世纪人才培养计划、江苏省青蓝工程学术骨干培养计划。

 

发言题目：基于模型的CPS设计与验证技术
发言内容：与软件设计与验证技术不同，CPS的建模和验证理论技术要能够对系统的非确定性行为进行规约，并可以对环境行为进行显性建模。当我们设计基于模型的CPS解决方案时，需要强调两条理论：（1）架构设计：一项复杂的任务可以经由简单构件组装形成，相反地，一个设计问题可以被分解成为几个简单的子问题。（2）基于需求的设计：正确的需求不但能够被准确规格化，并且，在设计阶段还可用来指导设计的选择和调试。因此，要实现上述理论要求，只有针对CPS的相关理论方法开展研究，如括模型驱动的设计原理、并发理论、分布式算法、形式化的系统规约和验证、控制理论、实时系统和混成系统等，将现有的从不同侧面对信息-物理融合系统进行描述的理论技术进行完善、改造和集成。
